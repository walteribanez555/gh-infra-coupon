# Frontend Module

This module provisions the infrastructure for hosting a static frontend web application using AWS S3 and CloudFront. It is designed to securely deliver web assets with optimized caching and CORS support.

## Features
- Creates an S3 bucket for static website hosting.
- Configures bucket ownership, public access blocking, and website settings.
- Sets up AWS CloudFront distribution for global content delivery.
- Integrates Origin Access Identity (OAI) for secure S3 access from CloudFront.
- Applies a bucket policy to restrict access to only CloudFront OAI.
- Enables CORS headers for frontend assets via CloudFront response headers policy.
- Custom error responses for SPA routing (index.html fallback).
- Manages tags for resource identification.

## Usage
Include this module in your Terraform configuration to deploy a secure, scalable, and performant static frontend for your application, served globally via CloudFront.
